# AWS-EKS-Based-DevOps-Automation-for-Microservices-based-E-Commerce-Application
End-to-end DevOps implementation of a microservices-based E-commerce application using Docker, Kubernetes, AWS EKS, Terraform, GitHub Actions, and Argo CD. Automated infrastructure, CI/CD pipelines, and deployment with GitOps principles, exposing services securely via Route 53 and Load Balancers.
